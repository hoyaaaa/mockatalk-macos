# Scripts

카카오톡Sub 생성과 업데이트 확인에 쓰는 스크립트입니다.

- `create-kakaotalk-macos-clone.sh`: 공식 `/Applications/KakaoTalk.app`을 복사해 `카카오톡Sub.app`을 만듭니다.
- `check-kakaotalk-sub-update.sh`: 공식 카카오톡과 `카카오톡Sub.app`의 버전을 비교하고, 다를 때만 업데이트 확인 창을 띄웁니다.
- `install-update-checker.sh`: 6시간마다 업데이트 확인 스크립트를 실행하는 사용자 LaunchAgent를 설치합니다.

비밀번호나 개인 토큰을 이 폴더에 넣지 마세요.
