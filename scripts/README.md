# Scripts

This directory is reserved for prototype scripts.

Suggested first scripts:

- `check-baseline.sh`: verify macSandbox baseline readiness
- `create-overlay.sh`: create persistent KakaoTalk overlay from baseline
- `run-qemu.sh`: run QEMU with RDP forwarding
- `connect-rdp.sh`: connect with external FreeRDP for early prototype

Do not put secrets here.
